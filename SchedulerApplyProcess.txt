1) Publish service

2) Copy at proper location (in server) - e.g D:\MAPIServices\OrderWorkerService

3) Verify settings/DB connection in appsettings.json

4) Install service by running below PowerShell commands

sc.exe create "AirmidTech_Schedule_Services" binpath= "F:\AirmidTech_HIMS_Services\AirmidTechScheduler_15Feb_v1\HIMS.ScheduleJobs.exe" start= auto

Set-Service -Name "AirmidTech Scheduler Services" -Description "AirmidTech Scheduler Service is designed to send automated Emails, WhatsApp notifications, and Daily PDF reports to the Management Team."


5) Start service

sc.exe start "AirmidTech HIMS Services"


6) Stop and Uninstall - incase need to re-installed updated/latest

sc.exe stop "AirmidTech HIMS Services"

sc.exe delete "AirmidTech HIMS Services"
